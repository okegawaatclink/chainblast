# 🎮 CHAIN BLAST — ゲーム仕様書 v1.0

## 概要

マウスで弾を引っ張って発射し、敵を「点火→爆発→連鎖」させるターン制カジュアルアクションゲーム。
敵は毎ターン上から1行ずつ落下し、最下段に到達するとゲームオーバー。
Wave 20をクリアすれば勝利。

---

## 技術仕様

- HTML + JavaScript（Canvas 2D）単一ファイル
- 外部ライブラリなし
- フィールドサイズ：560 × 500px
- フレームレート：requestAnimationFrame（可変dt）

---

## ターン構造

1ターンは以下のフェーズで構成される。

```
DROP（敵が1行落下）
 ↓
AIM（プレイヤーが狙いを定める）
 ↓
BULLET（弾が飛ぶ）
 ↓
CHAIN（連鎖が解決する）
 ↓
次のターンへ（DROP）
```

### フェーズ詳細

| フェーズ | 内容 |
|---|---|
| DROP | 新しい敵が上から落下、既存の敵も全員1行ずつ下がる（0.33秒アニメーション） |
| AIM | プレイヤーがドラッグして発射方向・強さを決める |
| BULLET | 弾が飛行・壁反射・敵ヒット処理 |
| CHAIN | 点火済みの敵が時間差で爆発し連鎖する |

---

## 操作

- **マウスドラッグ**：ランチャーからドラッグして離すと発射
  - ドラッグ距離で弾速が変わる（最低4、最大17 px/frame相当）
  - ドラッグ中はエイムライン（軌道予測）とパワーバーを表示
- **クリック**：GAME OVER / GAME CLEAR 画面でリスタート

---

## プレイヤー弾

| 項目 | 値 |
|---|---|
| 形状 | 円形（半径7px） |
| 壁反射 | あり（上下左右） |
| 消滅条件 | 壁に**3回**反射したら消滅 |
| 敵ヒット | 弾は消えない（何度でも当たれる） |
| 連続ヒット防止 | 同一敵への再ヒットは0.15秒クールダウン |
| 失速消滅 | 速度が0.4未満になると自動消滅 |
| ペナルティ | 1ターン中に**敵を1体も爆発できなかった場合**、次Waveの出現敵数が**1体**になる |

弾が消滅した時点でCHAINフェーズへ移行する。

## 敵

### 基本

| 項目 | 値 |
|---|---|
| 形状 | 円形（半径17px） |
| X座標 | ランダム配置（敵同士の間隔 ≥ 40px を保証） |
| Y座標 | グリッド行ベース（1行 = 40px） |
| 動作 | ターンごとに1行ずつ下降（プレイヤー操作中は静止） |

### 状態遷移

```
通常(n) → 点火(i) → 爆発(x) → 消滅(d)
```

| 状態 | 遷移条件 | 挙動 |
|---|---|---|
| 通常(n) | 弾が1回ヒット | 点火状態へ |
| 点火(i) | 弾が再ヒット or 爆発に巻き込まれる | 即爆発 |
| 点火(i) | 0.28秒経過 | 自動爆発 |
| 爆発(x) | 0.4秒経過 | 消滅(d) |

※Wave 1〜2のみ：点火した敵は爆発時に**追加で1回分の誘爆**が発生（安全導線）

---

## 連鎖システム

### 爆発の波及（誘爆）

- 爆発した敵の中心から半径 **約27px（ER×1.6）** 以内にいる敵を巻き込む
- 巻き込まれた敵が**通常**なら→点火状態へ
- 巻き込まれた敵が**点火済み**なら→即爆発（連鎖加速）

### 連鎖加速

- 各敵の点火ディレイ：`0.28 × 0.9^連鎖レベル` 秒
- 連鎖が深くなるほど爆発間隔が短縮される

### 連鎖カウント

- 爆発1回につき +1
- 画面上部に `N CHAIN` と表示
- 5連鎖以上で画面シェイク発生

## Waveシステム

| Wave | 出現数 |
|---|---|
| Wave 1 | 3体 |
| Wave 2 | 4体 |
| Wave 3 | 5体 |
| … | … |
| Wave 7以降 | 7体（最大、フィールド幅による上限） |

- 出現数：`min(2 + wave番号 + 連鎖ボーナス, 7)`
- 連鎖ボーナス：前Waveの連鎖数に応じて +0〜2（`floor(min(連鎖数,8)/3)`）
- 出現位置：毎回ランダムなX座標（重なり禁止）
- 出現方法：画面上部から1行目へアニメーション落下
- 既存の敵は全員1行下へシフト（同じアニメーションで同時）
- **Wave 20終了後：GAME CLEAR**
- ペナルティ発動時は、そのWaveの出現数が**1体固定**

## アイテム（選択ウィンドウ）

各Wave開始時に**アイテムを1つ生成**し、**爆発の誘爆**または**弾の直撃**で取得できた場合に**選択ウィンドウ**が出現する。
選択ウィンドウには**ランダムな2つの効果**が提示され、プレイヤーが1つを選ぶ。

### 生成ルール

- アイテムは**敵と同様にランダムX座標**へ配置（重なり禁止）
- **弾の直撃**または**爆発の誘爆**で取得
- 取得されなかったアイテムは**3ターン経過で消滅**

### 選択ルール

- 誘爆取得時に**ゲームを一時停止**してウィンドウを表示
- ウィンドウに**ランダムな2つの効果**を提示
- プレイヤーは**1つだけ選択**し、即時反映

### アイテム効果（例）

- `AIM GUIDE +10`：エイムガイドの予測点数を+10
- `BOUNCE +2`：弾の最大反射回数を+2
- `BLAST RANGE +0.2`：爆発半径倍率を+0.2
- `SCORE MULT +0.2`：スコア倍率を+0.2（加算式に乗算）

---

## ゲームオーバー条件

ターン終了後（CHAIN解決後）に、いずれかの敵が**最下段の行**に存在していたらゲームオーバー。
最下段はフィールド下端から80px上の位置。

---

## スコア

```
爆発1回ごとに加算：
  score += floor(10 × chainCount ^ 1.3 × scoreMultiplier)
```

- `chainCount` はそのターン中の累積爆発数
- `scoreMultiplier` 初期値は1.0、アイテムで+0.2
- ターン開始時（nextWave）にリセット
- BEST スコアはセッション中保持
